\# 仓库管理系统



\## 项目简介

本系统是一款面向中小型企业的仓库管理解决方案，支持商品入库、出库、库存盘点、批次管理及报表统计等核心功能，旨在通过数字化手段提升仓库运营效率，减少库存积压与缺货风险，降低人工管理成本。



\## 核心功能

\- \*\*基础信息管理\*\*：维护商品分类、供应商、仓库区域、计量单位等基础数据

\- \*\*入库管理\*\*：支持采购入库、退货入库、调拨入库等场景，记录批次、保质期、供应商等信息

\- \*\*出库管理\*\*：处理销售出库、领料出库、调拨出库等需求，关联订单信息并自动扣减库存

\- \*\*库存管理\*\*：实时展示库存余量，支持库存预警（低库存/超期提醒）、库存盘点与调整

\- \*\*报表统计\*\*：生成入库/出库明细、库存汇总、滞销商品分析等数据报表，支持导出Excel

\- \*\*用户权限\*\*：区分管理员、仓库员、操作员等角色，限制不同账号的操作权限

\## 快速开始

1\. 克隆项目到本地：`git clone \[你的仓库地址，如：https://github.com/xxx/warehouse-management.git]`

2\. 进入项目目录：`cd warehouse-management`

3\. 后端部署：

&nbsp;  - 导入 `sql/warehouse\_init.sql` 到数据库，初始化表结构与基础数据

&nbsp;  - 修改 `backend/src/main/resources/application.yml` 中的数据库连接配置

&nbsp;  - 执行 `mvn clean package` 构建Jar包，运行 `java -jar target/warehouse-1.0.0.jar` 启动服务

4\. 前端部署：

&nbsp;  - 进入前端目录：`cd frontend`

&nbsp;  - 安装依赖：`npm install`

&nbsp;  - 开发环境启动：`npm run dev`（访问 http://localhost:8080 ）

&nbsp;  - 生产环境构建：`npm run build`，将dist目录部署到Nginx
